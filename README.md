# Resume Analyzer & Job Matcher

## Overview

Resume Analyzer & Job Matcher is a Flask-based web application that helps users evaluate how well their resumes match a given job description. The system allows users to upload resumes in PDF format, analyzes the content, and provides keyword-based matching results and feedback.

---

* PDF Resume Upload
* Resume Text Extraction using PyPDF2
* Keyword-Based Matching

* Resume Feedback

## How It Works

1. User registers and logs in.
2. User uploads a resume in PDF format.
3. User enters a job description.
4. System extracts resume content.
5. Keywords from the job description are matched against the resume.
6. Match score and feedback are generated.
7. Results are displayed to the user.

## Installation

### Clone Repository

### Navigate to Project

cd resume-analyzer-job-matcher

### Create Virtual Environment

python -m venv venv

### Activate Environment

Windows:

### Install Dependencies

pip install -r requirements.txt

### Run Application

python app.py

### Open Browser

http://localhost:5000
